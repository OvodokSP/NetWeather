# NetWeather

**The Weather Forecast for Your Internet**

NetWeather — Android-приложение и виджет рабочего стола для мониторинга доступности интернет-ресурсов и диагностики сетевых ограничений.

## Возможности

- Индекс доступности сети 0–100%.
- Диагностика DNS → TCP → TLS → HTTP.
- Режимы: нормальный доступ, частичная деградация, вероятны ограничения, нет интернета.
- Российские, международные и пользовательские ресурсы.
- Ручное обновление на главном экране.
- Фоновый мониторинг через WorkManager.
- Интервалы 30 секунд, 1 минута, 5 минут, 15 минут.
- Для интервалов меньше 15 минут используется OneTimeWorkRequest с повторным планированием, не PeriodicWorkRequest.
- Виджеты 2x2, 4x2 и 4x4.
- История проверок.
- Уведомления о падении, восстановлении и медленном отклике.
- Экспорт/импорт заложены в архитектуру и могут быть расширены в следующей версии.

## Сборка

```bash
./gradlew assembleDebug
```

Если Gradle Wrapper ещё не создан:

```bash
gradle wrapper --gradle-version 8.7
./gradlew assembleDebug
```

## Архитектура

- `domain` — модели, анализатор доступности и бизнес-логика.
- `data` — Room, репозитории, сетевые проверки, StateStore для виджетов.
- `presentation` — Compose UI + ViewModel.
- `worker` — WorkManager и восстановление после перезагрузки.
- `widget` — Android AppWidgetProvider и RemoteViews.
- `notification` — уведомления.
